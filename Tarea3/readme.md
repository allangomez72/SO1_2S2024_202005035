## ⚙️ Configuración Inicial

Antes de iniciar el servicio, asegúrate de que todas las dependencias estén correctamente instaladas y configuradas.

```bash
cargo build --release
```

Otra cosa importante es compilarlo con el comando 
```bash
cargo run
```
